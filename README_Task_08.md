# Task 08 – Classification Models & Performance Metrics Pipeline

## Part A & B: Classification Modeling & Metrics Foundation
- **Logistic Regression Architecture:** Utilizes linear feature aggregation transformed by the Sigmoid function to output categorical event distribution boundary parameters.
- **Metric Definitions:** Accuracy gauges general reliability, Precision mitigates false positives, Recall checks missing target occurrences, and the F1-Score calculates the harmonic mean balance.

## Part C & D: Comparative Models Breakdown & Analysis
- **Decision Trees vs. Random Forests:** Individual branches scale linearly but risk overfitting to data noise. Random Forests aggregate these components across parallel estimators, balancing error rates via ensemble voting matrices.
- **Final System Recommendation:** The Random Forest Classifier delivers the best performance stability. It maintains an optimal balance across Precision and Recall constraints, outperforming simple regression algorithms on structured customer interaction metrics.
